支持的加速服务商

🚀 Jito

领先的 MEV 解决方案，支持交易加速和收益最大化

[provider.jito] enable = true tip = 1000

⚡ 0slot

高速交易服务，支持反 MEV 保护功能

[provider.zeroslot] enable = true anti-mev = true

🌊 Temporal

专业的交易加速服务，低延迟高成功率

[provider.temproal] enable = true tip = 1000000

🔷 NextBlock

新一代交易加速平台，支持前置运行保护

[provider.nextblock] enable = true front_running_protection = true

快速开始

1. 编译运行

# 调试模式运行 RUST_LOG=bot=debug,info cargo run # 生产模式编译 cargo build --release

2. 配置文件

# 复制配置模板 cp config.example.toml config.toml # 编辑配置文件 vim config.toml

3. Durable Nonce 配置

[base.durable_nonce] enable = true startup_update_nonce = false [[base.durable_nonce.account_config]] account = "VhE5EuzaC1Nyy4rEZ5hxzYCqHKP4eTtSqwz2PMzLf84" authority = "[your_authority_private_key]" payer = "[your_payer_private_key]"

4. 启动服务

# 运行主程序 ./bot run # 检查版本 ./bot version
